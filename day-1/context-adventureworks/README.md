# Tài liệu ngữ cảnh mẫu: dữ liệu AdventureWorks

Thư mục này là **ví dụ mẫu** cho phần "Tài liệu ngữ cảnh cho agent" (Block 4, Buổi 1). Đây là bộ
tài liệu tối thiểu mà agent đọc trước khi làm việc, giúp agent hiểu bối cảnh thay vì tự đoán.

Cấu trúc tối thiểu gồm ba mục:

- `01-nghiep-vu.md`: bối cảnh nghiệp vụ, doanh nghiệp sinh ra dữ liệu này làm gì.
- `02-du-lieu.md`: mô tả nguồn dữ liệu, ý nghĩa từng cột, các điểm cần lưu ý.
- `03-yeu-cau-bao-cao.md`: câu hỏi kinh doanh cần trả lời, ai đọc, quyết định gì.

Cách dùng trong buổi:
- **DEMO (giảng viên):** mở thư mục này, giải thích vai trò từng mục.
- **APPO 3 (học viên):** tạo một thư mục tương tự cho dự án của mình, mỗi mục viết vài dòng thật.

Nguồn dữ liệu: `../sales_adventureworks.csv`. Mọi mô tả trong `02-du-lieu.md` đã được đối chiếu với
dữ liệu thực tế (kiểm chứng), không suy đoán.
