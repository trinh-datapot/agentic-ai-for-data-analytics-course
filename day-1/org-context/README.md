# org-context/ — ngữ cảnh khai báo (declared context)

Thư mục này chứa **ngữ cảnh khai báo**: những điều "doanh nghiệp" cho chúng ta biết trước
khi bắt đầu. Agent đọc thư mục này để hiểu bối cảnh, thay vì tự suy đoán.

- `org-profile.md`: doanh nghiệp là ai, phân tích lĩnh vực gì, ngôn ngữ.
- `glossary.md`: các thuật ngữ kinh doanh, hiểu thống nhất một nghĩa.
- `sources.md`: danh mục nguồn dữ liệu (chi tiết từng cột nằm ở `knowledge/sources/`).

Phân biệt với `knowledge/`: `org-context/` là điều được **khai báo**, còn `knowledge/` là
**sự thật đã kiểm chứng** mà công việc tạo ra.
